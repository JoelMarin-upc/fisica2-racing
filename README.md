# Racing Game

## Description

This project recreates a racing game where you must finish the race first to win, trying to avoid the slow zones and use the nitros around the map.
The project uses the Box2D v2.4.1 library to simulate realistic physics, and it also uses the raylib v5.5.0 library to render graphics on the screen.

## Key Features

 - You control a fish.
 - Keep going forward & use the boosts around the map or even yours.
 - Avoid obstacles like the walls, jellyfish, or even other fish.
 - Be the first one to cross the winning line.
 - Choose your preferred map, difficulty, or laps.
 - If you lose, restart to play again at the moment.
 
## Controls

 ### Game controls
 - **LEFT ARROW** / **A**: Turn left
 - **RIGHT ARROW** / **D**: Turn right
 - **UP ARROW** / **W**: Curb / Go forward
 - **DOWN ARROW** / **S**: Curb / Go backward
 - **SPACE**: Nitro
 - **C**: Enables/Disables view rotation
 - **P**: Pause the game
 
 ### Debug controls
 - **F1**: Enables/Disables debug mode (Draws colliders, locks ball to mouse) & mouse joint on a character
